pymapmanager.interface.mmWindow module¶

This module defines MapManager window classes.

pymapmanager.interface.mmWindow: Base class for other windows to inherit from

pymapmanager.interface.mmStackWindow: Display and interact with the 3D image and annotations from a pymapmanager.mmStack.

pymapmanager.interface.mmStackPlotWindow: Display and interact with a plot of a annotations from a:class:pymapmanager.mmStack.

pymapmanager.interface.mmMapPlotWindow: Display and interact with a plot of pymapmanager.mmStack annotations accross a pymapmanager.mmMap.
