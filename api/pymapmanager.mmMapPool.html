pymapmanager.mmMapPool module¶

class pymapmanager.mmMapPool.mmMapPool(path)[source]¶

Load all maps in a folder.

Parameters:path (str) – Full path to a folder. This folder should contain folders of maps.

Example:

path = '/Users/cudmore/MapManagerData/richard/Nancy/'
maps = mmMapPool(path)
for map in maps:
        print map
